Projeto de banco de dados relacional desenvolvido para prática de SQL.

## Estrutura
- Cliente
- Produto
- Pedido
- ItemPedido

## Tecnologias
- MySQL

## Consultas
- Total por pedido
- Produto mais vendido
- Cliente que mais comprou

## Objetivo
Praticar modelagem de dados, relacionamentos e análise com SQL.
